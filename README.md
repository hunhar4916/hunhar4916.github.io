# Hunter Harwood's Portfolio

A personal portfolio website built with HTML and CSS as part of my Web Design coursework at ECPI University.

## Live Site

[https://hunhar4916.github.io](https://hunhar4916.github.io)

## About the Project

This project is a multi-page portfolio site that showcases my work and skills as a Software Development student. It was built from scratch using only HTML5 and CSS3.

## Features

- Responsive design with media queries for phone, tablet, and desktop
- Flexbox layout for navigation, main content, and contact form
- Project table with sample work using HTML table elements
- Contact form with accessible labels and input validation
- CSS transitions and hover effects throughout
- Hosted live via GitHub Pages

## Technologies Used

- HTML5
- CSS3
- GitHub Pages
